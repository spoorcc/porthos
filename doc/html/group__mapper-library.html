§ mapper_init()

int mapper_init ( float  x_size,
float  y_size,
unsigned int  max_level 
)

Initializes the mapper library.

Parameters
[in]x_sizeSize in X-direction
[in]y_sizeSize in Y-direction
[in]max_levelTwo to the max_level of detail

First mapper_init must be called to configure the library. With this call the real-world size that the map spans is set. The max_level sets the 2 to the Nth maximum level of detail. For example max_level of 2 means the map will be divided in 4x4 grid.
